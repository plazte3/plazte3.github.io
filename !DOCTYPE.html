Diccionario de Domótica

Automatización

Proceso de utilizar tecnología para realizar tareas de forma automática, sin intervención humana.

IoT (Internet de las Cosas)

Red de dispositivos físicos que se comunican e intercambian datos a través de internet.

Sensores

Dispositivos que detectan y responden a algún tipo de entrada del entorno físico.

Controlador

Dispositivo que gestiona y controla los sistemas domóticos en una vivienda.

Casa Inteligente

Hogar equipado con dispositivos conectados que permiten la automatización y el control remoto.

Actuadores

Dispositivos que reciben señales de control para realizar una acción física.

Escenarios

Configuraciones predefinidas de dispositivos domóticos que se activan para crear ambientes específicos.

Zigbee

Estándar de comunicación inalámbrica utilizado en la domótica para la conexión de dispositivos.

Red Mesh

Tipo de red donde los dispositivos se conectan entre sí, creando una malla de comunicación.

Alexa

Asistente virtual desarrollado por Amazon que controla dispositivos domóticos mediante comandos de voz.

Google Home

Dispositivo y asistente virtual de Google que permite controlar la domótica mediante comandos de voz.

IFTTT

Servicio que permite crear cadenas de condiciones, facilitando la integración y automatización de servicios y dispositivos.

Interruptores Inteligentes

Dispositivos que permiten controlar la iluminación de una casa de manera remota y automatizada.

Termostato Inteligente

Dispositivo que regula la temperatura de una vivienda de manera automatizada y eficiente.

Cerradura Inteligente

Sistema de bloqueo electrónico que se puede controlar mediante una aplicación móvil o comandos de voz.

Cámara IP

Dispositivo de videovigilancia que transmite señales de video a través de Internet.

Gateway

Dispositivo que actúa como un puente entre diferentes redes de comunicación en una casa inteligente.

Inteligencia Artificial

Tecnología que permite a las máquinas aprender y tomar decisiones basadas en datos.

Red Z-Wave

Protocolo de comunicación inalámbrica diseñado para la automatización del hogar.

Control por Voz

Tecnología que permite controlar dispositivos mediante comandos de voz.

Batería de Respaldo

Sistema de energía que proporciona electricidad en caso de un corte de energía.

Cerradura Biométrica

Cerradura que utiliza datos biométricos, como huellas digitales, para conceder acceso.

Enchufe Inteligente

Dispositivo que permite controlar el encendido y apagado de aparatos electrónicos de forma remota.

Detección de Movimiento

Tecnología que detecta el movimiento en un área específica para activar dispositivos.

Cámara de Seguridad

Dispositivo que monitorea y graba actividades para la seguridad del hogar.

Dispositivo Wearable

Tecnología portátil que puede integrarse con sistemas domóticos para monitorización y control.

Iluminación Ambiental

Sistema de iluminación que se ajusta automáticamente para mejorar el ambiente de una habitación.

Asistente Virtual

Software que ayuda a los usuarios mediante comandos de voz para realizar tareas y controlar dispositivos.

Sistema de Alarma

Conjunto de dispositivos diseñados para detectar intrusiones no autorizadas y alertar a los usuarios.

Red Inalámbrica

Tecnología que permite la comunicación entre dispositivos sin necesidad de cables.

Automatización de Iluminación

Sistema que permite el control automático de las luces en una vivienda.

Sistema de Audio Multi-Room

Tecnología que permite reproducir música en diferentes habitaciones de manera sincronizada.

Cámara de Timbre

Dispositivo que combina una cámara de seguridad con un timbre, permitiendo ver quién está en la puerta.

Monitoreo Remoto

Capacidad de supervisar y controlar dispositivos domóticos desde una ubicación distante.

Red Local

Red de comunicación que conecta dispositivos dentro de una casa o edificio.

Cierre Automático

Sistema que permite cerrar puertas y ventanas automáticamente por seguridad o comodidad.

Electrodomésticos Inteligentes

Aparatos que pueden ser controlados y monitorizados de forma remota para mayor eficiencia.

Control Climático

Sistema que gestiona la calefacción, ventilación y aire acondicionado de una vivienda.

Alarma de Humo Inteligente

Dispositivo que detecta humo y envía alertas a los usuarios a través de sus teléfonos.

Cámara de Interior

Dispositivo de seguridad para monitorear el interior de la vivienda.

Ventanas Inteligentes

Ventanas que pueden ser controladas automáticamente para abrirse o cerrarse.

Sistema de Riego Automático

Sistema que controla el riego del jardín de forma automática y eficiente.

Detector de Gas

Dispositivo que detecta fugas de gas y alerta a los usuarios.

Automatización de Cortinas

Sistema que permite abrir y cerrar cortinas de forma automática o remota.

Cámara Exterior

Dispositivo de seguridad para monitorear el exterior de la vivienda.

Control por Gestos

Tecnología que permite controlar dispositivos mediante gestos con las manos.

Sistema de Energía Solar

Tecnología que utiliza paneles solares para generar electricidad en una vivienda.

Automatización de Puertas

Sistema que permite abrir y cerrar puertas automáticamente por comodidad o seguridad.

Sensor de Humedad

Dispositivo que mide la humedad del aire y puede activar otros dispositivos en respuesta.

Automatización de Persianas

Sistema que permite controlar la apertura y cierre de persianas de forma automática.
